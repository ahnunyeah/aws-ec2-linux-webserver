
# My First AWS Cloud Project 

## Overview

This is my first hands-on AWS Cloud project.

In this project, I launched an Amazon EC2 instance running Amazon Linux 2023, connected to it securely using SSH from Windows PowerShell, installed and configured the Apache HTTP Server, and deployed a custom terminal-themed HTML website.

This project helped me understand the fundamentals of cloud computing, Linux server management, and web hosting on AWS.
